# Generic Shooter
A small concept game for a Python assignment made in a few days <br>
Just boot up and play :>

# Keybinds:
[SPACE] - Shoot <br>
[ESC] - Close <br>

[H] - Show Hitboxes <br>
[E] - Enable/Disable Asteroid Spawning <br>
[A] - Spawn an Asteroid <br>
[C] - Clear Screen Objects <br>
